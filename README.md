# Browser History Analysis

## Project Overview
This Digital Forensics project demonstrates how to analyze browser history artifacts from Google Chrome/Microsoft Edge using DB Browser for SQLite.

## Objectives
- Extract browser history
- Analyze visited URLs
- Review download history
- Identify search history
- Create a forensic timeline
- Prepare a professional forensic report

## Tools Used
- DB Browser for SQLite
- Google Chrome History Database
- Windows 11

## Evidence
Browser History SQLite database (History)

## Analysis Performed
- Visited URLs
- Google Search History
- Download History
- Timeline Reconstruction

## Screenshots
- URLs Table
- Downloads Table
- Search History

## Conclusion
Browser artifacts provide valuable evidence of user activity including visited websites, search queries, and downloaded files. Such artifacts are commonly used in Digital Forensics investigations.
